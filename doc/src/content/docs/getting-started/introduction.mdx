## What is Zudo Sg Docs?

Zudo Sg Docs is a documentation site built with [zudo-doc](https://github.com/zudolab/zudo-doc), a minimal documentation framework powered by zfb, MDX, and Tailwind CSS.

## Key Features

- MDX authoring with rich component support
- Fast static site generation via zfb
- Tailwind CSS v4 for styling
- Optional i18n, search, and more
