## Requirements

- Node.js 18 or later
- pnpm (recommended), npm, yarn, or bun

## Create a New Project

Run the scaffolding tool to create a new project:

```sh
pnpm create zudo-doc doc
```

## Start the Dev Server

```sh
cd doc
pnpm install
pnpm dev
```

Open [http://localhost:4323](http://localhost:4323) to view your docs.
